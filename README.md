# Electron Youtube-dl

## What you need (On PATH or in same directory)
* youtube-dl
* ffmpeg / ffprobe

## How to run?

1. `git clone`
2. `npm install`
3. `electron main.js`

### Install youtube-dl

    pip install -r requirements.txt

or on mac

    brew install youtube-dl

### Install ffmpeg

on mac

    brew install ffmpeg
